# Athlete Wearable Technology Analytics

## Overview

This project analyses athlete performance data collected from wearable technology sensors. The analysis focuses on identifying trends in running and cycling performance using data science techniques.

The goal was to explore training metrics and build predictive models that can support athlete performance monitoring.

## Analysis Performed

### Running Data Analysis
Exploratory data analysis was conducted on running activity data to understand:

- Pace statistics across training sessions
- Fastest and slowest running performances
- Monthly heart rate trends
- Heart rate zone classification
- VO2 max estimation

### Cycling Performance Modelling

Predictive models were developed to estimate **Functional Threshold Power (FTP)** for cycling performance.

Machine learning models explored include:

- Linear Regression
- Random Forest
- Gradient Boosting

Model performance was evaluated using **Mean Squared Error (MSE)**.

## Tools Used

- Python
- Pandas
- NumPy
- Jupyter Notebook
- Scikit-learn

## Key Insights

- Identified patterns in athlete training intensity through heart rate zone analysis.
- Analysed variations in pace and performance across training sessions.
- Developed predictive models capable of estimating cycling FTP using wearable sensor data.
